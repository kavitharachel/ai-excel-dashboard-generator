# AI Excel Dashboard Generator

## Overview
AI-powered application that converts Excel files into dashboards, insights, and SDLC documentation.

## Features
- Upload Excel files
- Auto-generate dashboards
- KPI detection
- AI insights
- FastAPI backend
- Lovable frontend

## Tech Stack
- Python (FastAPI)
- Pandas
- Lovable UI
